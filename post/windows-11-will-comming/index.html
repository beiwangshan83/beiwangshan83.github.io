Windows 11 它要来了？官方为何疯狂暗示

· 2021-06-08 ·

😎 我觉得你提醒的太明显啦

微软此前宣布，将于北京时间 6 月 24 日晚 23 点发布“下一代 Windows”，外界普遍预测为 Win11。

今日据外媒 Windows Latest，微软已经暂停了 Win10 DEV 预览的功能更新，新发布的 Build 21390.1000 没有任何新功能，只是为任务管理器、MSI 安装包启用了新的 Fluent 图标，允许 Windows Terminal Preview 作为默认终端模拟器，以及进行修复 Bug 的累积更新。

虽然微软前几年曾说 Windows 10将成为 Windows系统的最后一代，但现在它又想实现自我毁灭的承诺， CEO纳德拉月末将推出 Windows 11 (以下简称Win11)系统。

如今眼尖的网友又发现了官方的彩蛋，微软发布的新预告片中，微软 logo与视窗 log有夹角，有人量了一下，这个角度正好是11°，这被网友看作是微软官方的一个疯狂暗示。

不止如此，6月24日的发布会是在11点开始，这是一个很棒的决定，而且很明显是一个暗示。

在Win11系统方面，微软 CEO纳德拉说这是 Windows系统最重要的变化之一，而上周末高官又说新的 Windows系统和 Windows 95一样激动人心，无论如何，这意味着这是一个非常重要的系统，不大可能只是Win10在下半年的升级。
